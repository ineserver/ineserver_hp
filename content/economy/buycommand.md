職業ポイントを使って、冒険を便利にする機能や特別な権限を購入することができます。
これらはすべて **「買い切り」** タイプで、一度購入すれば永続的に使用可能です。

# 1. 購入方法

以下の機能は、サーバー内でコマンドを入力することで購入できます。

購入する際は、所持金が足りているか確認してください。

# 2. 販売リスト一覧

## 移動・テレポート系

| 機能名 | 価格 | コマンド | 解説 |
| :--- | :--- | :--- | :--- |
| **ホームポイント追加 (+1)** | **5,000 職業ポイント** | :command[/buycommandhome4] | :command[/sethome] で設定できるホームの数が合計**4つ**に増えます。 |
| **ワールド間移動機能** | **100,000 職業ポイント** | :command[/buycommandwtp] | ワールド間をテレポートで移動できる機能が解放されます。:command[/wtp <ワールド名>]で移動できるようになります |
| **無限飛行 (Fly)** | **1,000,000 職業ポイント** | :command[/buycommandfly] | サーバー内で空を飛ぶことができるようになります。建築や移動に最適です。:command[/tf]で切り替えができます。 |

## 便利ツール系

どこにいても作業ができる便利なポータブル機能です。

| 機能名 | 価格 | コマンド | 解説 |
| :--- | :--- | :--- | :--- |
| **ポータブル作業台** | **10,000 職業ポイント** | :command[/buycommandcrafting] | :command[/craft]でいつでも作業台を開けるようになります。 |
| **ポータブルエンダーチェスト** | **50,000 職業ポイント** | :command[/buycommandenderchest] | :command[/ec]でいつでもエンダーチェストを開けるようになります。 |
| **ポータブル金床** | **300,000 職業ポイント** | :command[/buycommandanvil] | :command[/anvil]でいつでも金床を開けるようになります。 |
| **ポータブルエンチャント** | **80,000 職業ポイント** | :command[/buycommandenchant] | :command[/enchant]でいつでもエンチャントテーブルを開けるようになります。 |

## 装飾・建築系

画像を貼り付けられる「画像地図」の設置上限を解放します。

画像地図の詳細については、[画像地図の使い方](/life/imageflame)をご覧ください。

| 機能名 | 価格 | コマンド | 解説 |
| :--- | :--- | :--- | :--- |
| **画像地図ライセンス (100枚)** | **50,000 職業ポイント** | :command[/buycommandframe100] | 画像地図の作成上限を**100枚**まで解放します。 |
| **画像地図ライセンス (1000枚)** | **150,000 職業ポイント** | :command[/buycommandframe1000] | 画像地図の作成上限を**1000枚**まで解放します。 |

# 3. 職業レベルによる割引システム

各機能には、特定の職業レベルが**20以上**の場合、購入価格が**半額**になる割引システムがあります。

| 機能名 | 割引対象職業 |
| :--- | :--- |
| **ホームポイント追加** | 冒険家(Adventurer) / 鉱夫(Miner) |
| **ワールド間移動機能** | 整地師(Digger) / 錬金術師(Alchemist) |
| **無限飛行** | 建築家(Builder) / 木こり(Woodcutter) |
| **ポータブル作業台** | 鍛冶屋(Smith) / 鉱夫(Miner) |
| **ポータブルエンダーチェスト** | 木こり(Woodcutter) / 農家(Farmer) |
| **ポータブル金床** | 鍛冶屋(Smith) / 整地師(Digger) |
| **ポータブルエンチャント** | 錬金術師(Alchemist) / 冒険家(Adventurer) |
| **画像地図ライセンス** | 建築家(Builder) / 農家(Farmer) |

**注意点**
* 一度購入した機能の払い戻しはできません。
