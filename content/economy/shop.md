アイテムが空中に浮いて見える、綺麗で分かりやすいショップの作り方を解説します。

# 1. ショップ開設の準備

ショップを作るには、専用のアイテム **「ショップ基礎アイテム」** が必要です。
以下のコマンドを入力して購入してください。

:::command
/ds buy
:::

* **費用:** 250 ine
* **アイテム名:** ショップ基礎アイテム

# 2. ショップの作成方法

1.  購入した **「ショップ基礎アイテム」** を、ショップを作りたい場所に設置します。
2.  設置すると、基礎ブロックの上に「空のショップ」という文字とパーティクルが表示されます。
3.  これでショップの枠組みは完成です！

# 3. 商品と価格の設定（編集メニュー）

ショップのオーナーは、ショップに向かって **【スニーク（Shift）＋右クリック】** することで編集メニューを開けます。

| アイコン | 機能説明 |
| --- | --- |
| **<span style="white-space: nowrap;">アイテム</span>** | 手持ちのアイテムをクリックしてセットすると、それが商品になります。 |
| **<span style="white-space: nowrap;">購入価格</span>** | 他の人が「買う」時の値段です。左クリックで増加、右クリックで減少。 |
| **<span style="white-space: nowrap;">売却価格</span>** | 他の人がこの店に「売る（買取）」時の値段です。※`-1` に設定すると買取を無効化できます（デフォルト）。 |
| **<span style="white-space: nowrap;">1スタックの量</span>** | 1回の取引で売買する個数を設定します。（例：1個ずつ売るか、64個まとめて売るか） |
| **<span style="white-space: nowrap;">在庫量</span>** | 販売や買取をしている商品の在庫数を管理できます。 |
| **<span style="white-space: nowrap;">残高</span>** | 売上金や物々交換で受け取ったものを回収したりします。（デフォルトでは、そのまま/moneyに入るようになります。） |
| **<span style="white-space: nowrap;">通貨タイプ</span>** | 取引に使用する通貨を変更します。（通常のお金、アイテム同士の物々交換など |
| **購入数制限・売却数制限（グローバル）** | ショップ全体での取引総数の上限を設定します。（「先着〇〇個限定」のように、売り切りの設定ができます） |
| **購入数制限・売却数制限（1プレイヤーあたり）** | プレイヤー1人あたりの取引上限数を設定します。（「お一人様〇個まで」のような制限がかけられます） |


# 4. ショップを削除したい場合

ショップを削除するには、オーナーが　**スニークしながらショップを破壊（左クリック）**　してください。

在庫アイテムや「ショップ基礎アイテム」はドロップして戻ってきます。

なお、ショップ内に在庫や残高があると削除できませんのでご注意ください。
