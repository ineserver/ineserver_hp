サーバー内で用意されている「分譲地」などの土地を購入する方法を解説します。
土地を購入すると、その場所で自由に建築ができるようになります。
一度購入すれば、維持費などはかからず永続的にあなたの土地になります。

# 1. 販売されている土地の見分け方

販売中の土地には看板が立っています。
看板の1行目が <span style="color: green;">**販売中**</span> となっている場所が購入可能です。

**看板の見方**
* **1行目:** <span style="color: green;">**販売中**</span> （この表記なら購入可能）
* **2行目:** 土地の名前（ID）
* **3行目:** 価格（例: `1000 ine`）
* **4行目:** 土地の広さ（例: `10x10x10`）


# 2. 購入方法

土地にある看板を**右クリック**するだけで購入できます。
（チャット欄に確認メッセージが出た場合は、指示に従ってクリックまたはコマンド入力してください）

所持金（ine）が足りていれば、すぐにあなたの土地になります。

# 3. 土地の管理メニュー (ARM GUI)

購入した土地の管理は、すべて専用のメニュー画面から行えます。
以下のコマンドを入力してください。

:::command
/arm gui
:::

このメニューからは以下の操作が可能です。

* **購入した土地 (オーナー):** 自分が持っている土地の一覧を表示します。
* **所属している土地 (メンバー):** 友達の土地など、メンバーとして参加している土地を表示します。
* **土地検索:** 空いている土地を探すことができます。

## 自分の土地の設定
メニューから自分の土地アイコンをクリックすると、詳細設定画面が開きます。

* **土地へテレポート:** その場所へ瞬時に移動します。
* **メンバー管理:** 友達に建築権限を与えたり、削除したりできます。
* **土地のリセット:** 建築物をすべて消去して初期状態に戻します（※元に戻せないので注意！）。
* **土地を売却 (Sellback):** 土地を手放して、サーバーに返却します（設定された％のお金が戻ってきます）。

# 4. メンバーの追加（友達と住む）

友達と一緒に住みたい場合は、以下のコマンドを使うか、GUIメニューから設定します。

:::command
/arm addmember <土地名> <プレイヤー名>
:::

例：土地 `region1` に `nekodesu` さんを追加する場合
```

/arm addmember region1 nekodesu

```
※相手がサーバーにログインしている必要があります。
