経験値をボトルに詰めてアイテム化（保存）する機能について解説します。

この機能を使えば、デスポーンによる経験値ロストを防いだり、プレイヤー間で経験値を取引したりすることができます。

# 1. 経験値ボトルの基本

通常、経験値はプレイヤー自身に蓄積されますが、この機能を使うことで**アイテムとして持ち運ぶ**ことができます。

* **デスペナルティ対策:** 危険な冒険に行く前にボトルに入れてチェストに預けておく。
* **取引:** 「経験値 1000」などをアイテムとして他人に渡す・売る。

# 2. 経験値をボトルに入れる

経験値をアイテム化するには、以下のコマンドを使用します。

## 通常の経験値瓶に変換する
自分の持っている経験値を、Minecraftバニラの「エンチャントの瓶」に変換します。

:::command
/bottle get <数値 または max>
:::

* **:command[/bottle get 10]**: 経験値を消費して、ボトルを1本取り出します（例）。
* **:command[/bottle get max]**: 持っている全ての経験値をボトルに変換します。

## 数値を指定して保存する
指定した経験値量を、**1つのボトルにまとめて**保存します。

大量の経験値を管理したい場合や、特定量の取引に便利です。

:::command
/bottle store <数値 または max>
:::

作成されたボトルには「保存してある経験値: 〇〇」という説明書き（Lore）がつきます。

このボトルを使用（右クリック）すると、記載された分の経験値を獲得できます。

# 3. 便利なツールコマンド

経験値管理に役立つ計算・確認コマンドです。

## ステータス確認
現在のレベルや、次のレベルまでに必要な経験値量を詳しく表示します。

:::command
/bottle stats
:::

## 目標レベルまでの計算
指定したレベルに到達するまでに、あとどれくらいの経験値（またはボトル）が必要かを計算します。

:::command
/bottle until <目標レベル>
:::
例：レベル30になるのに必要な量を知りたい場合 → :command[/bottle until 30]

# 4. 仕様・変換レート

サーバーでの設定値は以下の通りです。

| 項目 | 設定値 | 解説 |
| :--- | :--- | :--- |
| **変換コスト** | **10 XP** | ボトルを作るために消費される経験値 |
| **獲得量** | **10 XP** | ボトルを割った時に得られる経験値 |
| **手数料** | **なし** | 変換時にお金はかかりません |
| **最低経験値** | **7 XP** | ボトルを作るには最低 7 XP必要です |

::::details{title="変換ロスについて"}
いねさばの設定では、**消費する経験値と、獲得できる経験値が同じ（10 XP）** に設定されています。

つまり、ボトルに詰め替えることによる経験値の損失（ロス）はありません。安心して変換してください。
::::

# 5. 注意点
* コマンド入力時、インベントリに空きがあることを確認してください。
* **`/bottle store`** で作成した特殊なボトルは、投げることで経験値に戻ります。マグマなどに投げ込んで消滅させないようご注意ください。
