いねさばでは、相手がログアウトしていてもメッセージやアイテムを送れる便利な**メールシステム**を導入しています。

「友達にアイテムを渡したいけど時間が合わない…」という時に最適です！

# 1. メールの基本

メールシステムはすべてコマンド、またはGUI（メニュー画面）で操作します。

## 受信箱を見る
届いたメールを確認するには、以下のコマンドを入力します。

:::command
/mail inbox
:::


未読メールがある場合、サーバーに参加したタイミングでお知らせが表示されます。

## メールを送る（作成）
メールを新しく作成するには、以下のコマンドを入力します。

:::command
/mail write
:::


このコマンドを入力すると、**「メールの編集」** 画面が開きます。
以下の手順でメールを作成しましょう。

1.  **[宛先]** アイコンをクリックし、送りたいプレイヤー名を入力します。
2.  **[M1]** アイコンをクリックし、メッセージを入力します。
3.  準備ができたら **[送信する]** アイコンをクリックして完了です！

::::details{title="コマンドだけで手軽に送りたい場合"}
短いメッセージだけを送りたい場合は、以下のコマンドで1行メールを即座に送信できます。

:::command
/mail text <相手の名前> <メッセージ>
:::

::::

# 2. アイテムの添付・代引き取引

メールにはメッセージだけでなく、アイテムやお金を添付することができます。

## アイテムを送る
メール編集画面にある **[添付ボックスを開く]**（チェストのマーク）をクリックすると、アイテムを入れる画面が開きます。

ここに送りたいアイテムを入れてください。

## 代引き（着払い）機能
「アイテムを渡す代わりに、お金をもらいたい」
そんな時に便利なのが、**着払い設定**です。

1.  アイテムを添付します。
2.  メール編集画面の **[着払い料金を設定する]** をクリックします。
3.  請求したい金額（ine）を入力します。

これを受け取った相手は、**設定された金額を支払わないとアイテムを取り出せない**仕組みになっています。

安全なアイテム取引に活用してください！

# 3. グループ機能

チームや町のメンバーなど、よくメールを送る相手をグループとして登録できます。

:::command
/ugroup
:::


このコマンドでグループ管理画面が開き、メンバーの追加や一斉送信が可能になります。

作成できるグループは1人 **5つ** まで、1グループあたり **15人** まで登録可能です。

# 4. 注意事項

<span class="text-red">**メールの保存期間について**</span>

メールの保存期間は **30日間** です。

送信から31日が経過したメールは**自動的に削除**されますので、添付アイテムなどの取り出し忘れにご注意ください。

* **ゴミ箱:** 削除したメールは一時的にゴミ箱に入ります。:command[/mail trash] で確認・復元が可能です。
* **スパム対策:** メール送信後、次のメールを送るまで **15秒** の待機時間が必要です。
