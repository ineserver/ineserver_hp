ごちゃごちゃになってしまったチェストやインベントリを、一瞬で綺麗に整頓してくれる便利な機能「自動整理」について解説します。

この機能を使えば、アイテムの種類ごとに並び替えたり、チェストへの補充を一括で行ったりすることができます。

# 1\. 基本的な使い方

サーバーに参加した時点で、基本的な整理機能は有効になっていますが、設定メニューから自分好みにカスタマイズが可能です。

以下のコマンドを入力すると、設定メニュー（GUI）が開きます。

:::command
/sort
:::
または :command[/chestsort] でも開くことができます。

# 2\. 設定メニューの解説

メニュー内の各アイコンをクリックすることで、機能の **有効（緑）** / **無効（赤）** を切り替えることができます。

| アイコン名 | 機能説明 |
| :--- | :--- |
| **自動整理** | チェストや樽などを**開いたとき**に、自動で中身を整理します。 |
| **インベントリ自動整理** | 自分のインベントリを**開いたとき**に、自動で整理します。 |
| **ブロックを左クリック** | チェストや樽などを**叩く（左クリックする）** だけで、開かずに中身を整理します。 |

# 3\. 便利なショートカット（ホットキー）

インベントリを開いている最中に、特定の操作をすることで「その場ですぐに整理」をすることができます。これらはメニュー内のアイコンで個別にON/OFFが可能です。

| 操作 | 動作 |
| :--- | :--- |
| **ミドルクリック** | 任意のスロットをマウスホイールでクリックすると整理します。 |
| **Shift + クリック** | 空きスロットをShiftキーを押しながらクリックすると整理します。 |
| **ダブルクリック** | 空きスロットをダブルクリックすると整理します。 |
| **Shift + 右クリック** | 空きスロットをShiftキーを押しながら右クリックすると整理します。 |

# 4\. 一括補充・回収機能（上級者向け）

チェストを開いた状態で、**インベントリの枠外（GUIの外側）** をクリックすると発動する便利な機能です。建築資材の補充などに非常に役立ちます。

## チェストへ補充（左クリック）

インベントリの枠外を **左クリック** します。

  * **シングルクリック:** チェスト内にあるアイテムと同じものを、手持ちから補充します。
  * **ダブルクリック:** ホットバー以外の<span class="text-red">**全アイテム**</span>をチェストに詰め込みます。

## チェストから回収（右クリック）

インベントリの枠外を **右クリック** します。

  * **シングルクリック:** 手持ちにあるアイテムと同じものを、チェストから回収します。
  * **ダブルクリック:** チェスト内の<span class="text-red">**全アイテム**</span>をインベントリに回収します。

::::details{title="コマンドでの操作"}
設定メニューを開かずに、コマンドだけで手持ちのアイテムを整理することも可能です。

:::command
/invsort
:::
または :command[/isort]

  * **:command[/invsort on]**: インベントリの自動整理を有効にします。
  * **:command[/invsort off]**: インベントリの自動整理を無効にします。
  * **:command[/invsort toggle]**: 有効/無効を切り替えます。
::::
