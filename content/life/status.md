いねさばでは、他のプレイヤーに「今の自分の状態」を伝えるための**ステータス機能**が導入されています。

「作業に集中したい」「今は離席中」「雑談したい！」といった状態をTABに表示できます。

# 1. ステータスの変更方法

ステータスの変更は、専用のメニュー画面から簡単に行えます。

以下のコマンドを入力してください。

:::command
/status
:::

メニューが開いたら、設定したい状態のアイコンをクリックするだけで適用されます。

## ステータス一覧

メニューで選択できるステータスは以下の通りです。

| アイコン | ステータス名 | こんな時に使おう |
| :--- | :--- | :--- |
| **看板** | **雑談歓迎** | 誰かと話したい、チャットを楽しみたい時に。 |
| **赤色のベッド** | **離席中 (AFK)** | トイレやご飯などで、少し画面の前から離れる時に。 |
| **鉄のツルハシ** | **作業中** | 建築や整地に集中していて、手が離せない時に。 |
| **望遠鏡** | **撮影中** | 動画の撮影中や、スクショ撮影に集中している時に。 |
| **ネコのスポーンエッグ** | **ねこ** | にゃーん（気まぐれに過ごしたい時に）。 |
| **バリア** | **リセット** | 通常の状態に戻します。 |

※ステータスを設定すると、TABリスト（プレイヤー一覧）にステータス名が表示されます。

# 2. 離席モード (AFK) の便利な機能

「離席中 (AFK)」には、他のステータスにはない機能があります。

## 自動AFK機能
操作をしないまま **5分** が経過すると、自動的に「離席中」ステータスになります。

少し動くだけで自動的に解除され、元のステータスに戻ります。

## 理由付きAFKとパーティクル
コマンドを使って離席理由を入力すると、その理由に合わせて**頭上のエフェクト（パーティクル）が変化**します！

状況を楽しく伝えることができるので、ぜひ使ってみてください。

**コマンド入力例:**
:::command
/afk ご飯
:::
※または :command[/status afk お風呂] など

# 3. コマンド一覧

メニューを開かずに直接コマンドで切り替えることも可能です。

* **メニューを開く:** :command[/status]
* **離席する:** :command[/afk] または :command[/afk <理由>]
* **作業中モード:** :command[/status work]
* **撮影中モード:** :command[/status rec]
* **雑談歓迎モード:** :command[/status chat]
* **ねこモード:** :command[/status cat]
* **解除:** :command[/status off]

::::details{title="コマンドのエイリアス（省略形）について"}
一部のコマンドは、英語入力でも日本語入力でも反応するように設定されています。

* `/status 作業中`
* `/status 撮影中`
* `/status 雑談`
* `/status ねこ`

これらでも変更が可能です。
