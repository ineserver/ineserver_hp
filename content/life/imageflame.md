インターネット上の画像をマインクラフト内の「地図」として貼り付けられる機能の使い方を解説します。

# 1. はじめに（重要なルール）

画像を使用する際は、以下のルールを絶対に守ってください。

<div style="border: 2px solid red; padding: 10px; background-color: #fff0f0;">
<h3 class="text-red" style="margin-top: 0;">⚠️ 画像アップロードに関する警告</h3>

**著作権や肖像権を侵害する画像、および公序良俗に反する画像の使用は固く禁止されています。**

違反した場合、画像地図の撤去だけでなく、サーバーからの処罰対象となる場合があります。法律とマナーを守って使用してください。
</div>

# 2. 画像地図の作成方法

インターネット上の画像（URL）を使って、ゲーム内で設置できるアイテムを作成します。

## 手順

1.  **画像のURLを用意する**

    DiscordやImgur、Gyazoなどに画像をアップロードし、その「画像アドレス（URL）」をコピーします。

    ※URLは `http` または `https` で始まり、末尾が `.png` や `.jpg` など画像形式で終わるものである必要があります。

2.  **作成コマンドを入力する**

    以下のコマンドを入力して地図を作成します。

    :::command
    /if create <地図の名前> <URL> <幅> <高さ>
    :::

    * **地図の名前**: 自分で管理しやすい名前を付けてください。

    * **URL**: 手順1で用意した画像のURL。

    * **幅・高さ**: 画像を何マスの大きさにするか指定します（例: `2 1` なら横2マス、縦1マス）。

3.  **設置する**

    コマンドが成功すると、インベントリに地図アイテムが入ります。

    壁に **額縁** を設置し、その額縁に向かって地図アイテムを持って **右クリック** すると画像が貼り付けられます。

# 3. 作成上限とライセンスの購入

サーバー負荷軽減のため、1人が作成できる画像地図の枚数には上限があります。

初期状態では **10枚** まで作成可能です。

もっとたくさんの画像を飾りたい場合は、以下のコマンドから追加ライセンスを購入することで上限を増やせます。

| ライセンス名 | 枚数 | 価格 | コマンド |
| :--- | :--- | :--- | :--- |
| **画像看板ライセンス (100枚)** | 100枚 | 50,000 ine | :command[/buycommandframe100] |
| **画像看板ライセンス (1000枚)** | 1000枚 | 150,000 ine | :command[/buycommandframe1000] |

※購入は「買い切り」で、一度買えば永続的に有効です。

# 4. 便利な機能

建築に役立つ便利な機能がいくつか用意されています。

## 透明な額縁の作り方
画像を壁に直貼りしているように見せたい場合、「透明な額縁」が便利です。

いねさばでは、サバイバルモードでも以下の方法で透明な額縁を作成できます。

1.  通常の額縁を地面にドロップする。

2.  **「透明化のポーション（スプラッシュ）」** を額縁に投げつける。

3.  額縁が透明になります！

::::details{title="透明額縁の詳細仕様"}
* 一度に最大 **8個** までの額縁を透明化できます。
* 透明な額縁の中にアイテムがない場合、うっすらと発光して場所がわかるようになっています（アイテムを入れると完全に消えます）。
::::

## 作成済み地図の管理
自分が作成した地図の一覧を確認したり、削除したりするには以下のコマンドを使用します。

* **一覧を表示**: :command[/if list]
* **地図を削除**: :command[/if delete <地図の名前>]
