サーバー内での建築物を保護するために、土地の保護（Region）を設定する方法を解説します。

# 1. 準備するもの

土地の範囲を選択するために、**__木の棒__**を使用します。

# 2. 保護の手順

## ① 範囲を選択する
**木の棒**を手に持ち、保護したいエリアの対角線上の2点をクリックします。

1.  **左クリック** で始点（1点目）を指定
2.  **右クリック** で終点（2点目）を指定

::::details{title="範囲を垂直方向に拡張する（おすすめ）"}
範囲を選択した後、以下のコマンドを入力すると、岩盤から空の高さまで垂直方向に範囲を拡張できます
:::command
//expand vert
:::
::::

## ② 保護領域を作成する
範囲選択ができたら、以下のコマンドを入力して保護を作成します。


:::command
/rg create <自分の名前_保護名>
:::

<span class="text-red">**必ず、保護名は「自分の名前_保護名(保護名は自由)」にしてください！**</span>（保護名の重複を避けるためです）

例：
`
/rg create 1necat_myhome
`

これで、自分以外のプレイヤーはその場所を編集できなくなります。

::::details{title="選択した範囲のパーティクルを消す方法"}
木の棒で選択した範囲を示すパーティクルを消すには以下のコマンドを入力してください。
:::command
/wesv toggle
:::

なお、パーティクルを再び出したい場合も同じコマンドを打ってください。
::::

# 3. メンバーの追加（友達と共有する）

友達にも建築の権限を与えたい場合は、メンバーに追加します。

:::command
/rg addmember <保護名> <プレイヤー名>
:::

メンバーから削除する場合：
:::command
/rg removemember <保護名> <プレイヤー名>
:::

# 4. 保護の削除

作成した保護を削除したい場合は、以下のコマンドを使います。

:::command
/rg remove <保護名>
:::

# 5. 保護情報の確認

現在いる場所の保護情報を確認するには、以下のコマンドを使います。

:::command
/rg i
:::

## WorldGuard フラグ一覧
保護情報の中に「フラグ(Flags)」を選択できる場所があります。いねさばで変更できるフラグは以下のとおりです。

| フラグ名 | 説明 |
| :--- | :--- |
| **【操作・アクション】** | |
| `use` | ドア、ボタン、レバー、感圧板などの基本的なブロックの使用 |
| `interact` | ブロックやエンティティへの右クリック操作全般（`use`よりも範囲が広い） |
| `chest-access` | チェスト、かまど、樽などのインベントリを持つブロックへのアクセス|
| `use-anvil` | 金床の使用（修理や名前変更） |
| `use-dripleaf` | 大きなドリップリーフに乗った際、葉が傾くかどうかの挙動 |
| `item-frame-rotation` | 額縁に飾ってあるアイテムを回す操作 |
| `damage-animals` | 牛や羊などの友好MOBへの攻撃・ダメージ |
| `pvp` | プレイヤー同士の攻撃・ダメージ |
| `enderpearl` | エンダーパールの使用およびテレポート |
| `chorus-fruit-teleport` | コーラスフルーツを食べた際のテレポート |
| **【乗り物】** | |
| `ride` | 馬、豚、トロッコ、ボートなどの乗り物への乗車 |
| `vehicle-place` | トロッコやボートなどの乗り物の設置 |
| `vehicle-destroy` | トロッコやボートなどの乗り物の破壊 |
| **【環境・成長】** | |
| `mob-spawning` | モブの自然スポーン|
| `crop-growth` | 小麦、ジャガイモ、ニンジンなどの農作物の成長 |
| `mushroom-growth` | キノコの成長や巨大キノコ化 |
| `mycelium-spread` | 菌糸ブロックの土ブロックへの伝播 |
| `vine-growth` | ツタ（つる）の成長 |
| `rock-growth` | 鍾乳石（ドリップストーン）の成長 |
| `sculk-growth` | スカルク系ブロック（スカルクカタリストによる侵食）の広がり |
| `snow-fall` | 天候による雪の積もり（レイヤー生成） |
| `snow-melt` | 積もった雪の自然溶解|
| `frosted-ice-form` | 氷結歩行エンチャントによる氷の生成 |
| `frosted-ice-melt` | 氷結歩行または自然生成された氷の溶解 |
| **【メッセージ】** | |
| `greeting` | そのエリアに入った時にチャット欄に表示されるメッセージ |
| `greeting-title` | そのエリアに入った時に画面中央（タイトル）に表示されるメッセージ |
| `farewell` | そのエリアから出た時にチャット欄に表示されるメッセージ |
| `farewell-title` | そのエリアから出た時に画面中央（タイトル）に表示されるメッセージ |
