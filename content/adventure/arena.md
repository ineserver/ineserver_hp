迫りくるモンスターの波（ウェーブ）を耐え抜き、報酬を獲得する「アリーナ」の解説です。

一人で腕試しをするもよし、友達と協力して攻略するもよしのPvEコンテンツです。

# 1. 参加方法

アリーナに参加するには、以下のコマンドを入力してください。

:::command
/arena join <アリーナ名>
:::

または、短縮コマンド :command[/ma join <アリーナ名>] も使用可能です。

## ゲームの流れ
1.  **ロビー待機:** コマンドを入力するとロビーに転送されます。
2.  **クラス選択:** 剣士や銃使いなど、戦い方に合ったクラス（職業）を選びます。
3.  **戦闘開始:** 一定時間が経過するか、全員の準備が完了するとゲームスタート！
4.  **ウェーブ:** モンスターが次々と湧いてきます。すべて倒すと次のフェーズに進みます。
5.  **終了条件:** ライフが尽きる（通常3回死亡）か、制限時間を生き残れば終了です。

# 2. アリーナ一覧と特徴

現在稼働している2つのアリーナについて解説します。

## Village (村)
**【制限時間: 30分 / 最大5人】**

王道のサバイバルマップです。

リスポーン地点が固定（2箇所）されているため、倒されても味方との合流が簡単で、初心者の方でも遊びやすくなっています。

* **敵の特徴:** 基本的な**スケルトン**や**ゾンビ**が中心ですが、ウェーブが進むと数が増えて混戦になります。後半には**ガーディアン**も出現するため注意が必要です。

## Valley (谷)
**【制限時間: 20分 / 最大5人】**

高難易度のサバイバルマップです。

**「ランダムリスポーン」** が有効になっており、死亡して復活するたびにバラバラの場所に飛ばされます。味方とはぐれやすいため、個人の生存能力が試されます。

* **敵の特徴:** スケルトン・ゾンビに加え、空から**ファントム**が襲来します。さらに**ガスト**や**ブレイズ**、強力なボスクラスの敵「**SkeletonKing**」が出現する激しいウェーブがあります。

# 3. クラスと強化 (Tier)

アリーナ内では、独自の「クラス」を選択して戦います。

* **Sword (剣士):** 近接攻撃に特化した基本クラスです。Tierが上がるとネザライト装備やエンチャント（鋭さ・防護）が付与され、鉄壁の守りになります。
* **Gun (銃使い):** 遠距離から敵を狙い撃つクラスです。Tierによって使える銃が変わります。
    * **Tier 1：** .50 GS（ハンドガン）、コンバットナイフ
    * **Tier 2：** AUG（アサルトライフル）、AX-50（スナイパー）
    * **Tier 4：** RPG-7（ロケットランチャー）などが使用可能になります。

### クラスのアップグレード
アリーナで稼いだ「ナゲット」を使うことで、クラスの **Tier（ランク）** を上げることができます。

Tierが上がると装備が強力になるほか、移動速度上昇などのポーション効果が付与されることもあります。

# 4. 報酬システム (ナゲット)

アリーナで遊ぶと、専用通貨 **「ナゲット (Nugget)」** が手に入ります。

モンスターを倒したりウェーブを生き残ったりすることで経験値が貯まり、ゲーム終了後にナゲットに換金されます。

貯めたナゲットは、以下のコマンドで開くショップで使用します。

:::command
/arena rewards
:::

### 報酬ショップのラインナップ (一部)
以下のようなレアアイテムや便利アイテムと交換可能です。

* **エンチャントの瓶 x16** (16ナゲット)
* **新プレミアムガチャ** (500ナゲット) - サーバー内のガチャを回せるスカルアイテム
* **幸運 V (本)** (1,500ナゲット)
* **修繕 (本)** (3,000ナゲット)

# 5. 主なコマンド一覧

| コマンド | 説明 |
| :--- | :--- |
| :command[/arena join <名前>] | 指定したアリーナに参加します。 |
| :command[/arena leave] | 参加中のアリーナから退出（リタイア）します。 |
| :command[/arena list] | 利用可能なアリーナの一覧を表示します。 |
| :command[/arena rewards] | ナゲットを使って報酬を交換・クラス強化します。 |
