いねさばでの冒険を彩る「実績・称号システム」について解説します。

サーバー内で特定の条件（プレイ時間、スキルのレベル上げ、イベント参加など）を達成すると、 **「実績」** が解除されます。

実績を解除すると、その証として **「称号」** を獲得でき、自分の名前の上に表示して他のプレイヤーに自慢することができます！

# 1. 実績メニューの開き方

以下のコマンドを入力すると、実績システム専用のメニュー（GUI）が開きます。

:::command
/ach
:::

または、:command[/achievements] でも開くことができます。

# 2. メニューの見方と操作

メニューを開くと、実績の「カテゴリ選択画面」が表示されます。

* **常設実績:** いつでも挑戦できる基本的な実績です。
* **期間限定:** イベント時など、限られた期間だけ挑戦できる実績です。
* **その他:** バグ報告や特別な活動に対する実績です。
* **ロードマップ:** McMMOやJobsなどの段階的な成長目標です。

## 実績の詳細を確認する
カテゴリをクリックすると、実績の一覧が表示されます。

アイコンにカーソルを合わせると、達成条件や説明を確認できます。

* <span style="color: green;">**✔ 獲得済み**</span>：すでに達成している実績です。取得日が表示されます。
* <span class="text-red">**✘ 未獲得**</span>：まだ達成していない実績です。「あと〇〇」といった進捗状況が表示されます。

::::details{title="実績の並び替え（ソート）機能"}
実績一覧画面の左下にあるアイコンをクリックすると、表示順を切り替えることができます。

* **標準:** デフォルトの順序
* **取得日時:** 新しい順 / 古い順
* **レアリティ:** 高い順 / 低い順

見つけにくい時は並び替えを活用してみましょう！
::::

# 3. 称号の設定方法

解除済みの実績は、 **「称号」** として使用できます。

称号を設定すると、プレイヤーの頭上（名前の上）にカッコいい装飾付きのテキストが表示されます。

1.  :command[/ach] でメニューを開き、獲得済みの実績（アイコンが光っているものなど）を探します。
2.  そのアイコンを**左クリック**します。
3.  チャット欄に「称号を選択しました」と表示されれば設定完了です！

解除したい場合は、現在選択中の称号（★マークが付いています）をもう一度クリックしてください。

※ 現在付けている称号を確認したい場合は、:command[/menu]より、自分の頭にカーソルを合わせると、一番上に表示されます。

# 4. ロードマップ機能

「次はなにを目標にすればいいの？」と迷った時に便利なのが**ロードマップ**です。

McMMOのパワーレベルや、Jobsの総合ポイントなど、段階的な目標がすごろくのような見た目で表示されます。

**確認方法:**
1.  :command[/ach] でメニューを開きます。
2.  **「ロードマップ表示」**（地図のアイコン）をクリックします。
3.  確認したい項目（McMMOやJobsなど）を選択します。

現在の自分の数値と、次の目標までの残りが一目でわかります。

進捗に応じて色付きガラスの道が繋がっていくので、コンプリート目指して頑張りましょう！

# 5. その他の機能

## 他の人の実績を見る
気になるプレイヤーがどんな実績を持っているか確認することができます。

:::command
/ach view <プレイヤー名>
:::

例：`nekodesu` さんの実績を見たい場合
:command[/ach view nekodesu]

## 実績の種類（一部紹介）
サーバーには様々な実績が用意されています。

| カテゴリ | 内容の例 |
| :--- | :--- |
| **プレイ時間** | 10時間、100時間プレイするなど、サーバーでの生活時間の証です。 |
| **チュートリアル** | 初めての採掘、初めての作製など、初心者が最初に目指す実績です。 |
| **McMMO** | 「採掘王」「伐採王」など、各スキルのレベルに応じた実績です。 |
| **Jobs** | 職業レベルや稼いだポイントに応じた実績です。 |
| **投票** | サーバーへの投票回数に応じた実績です。 |
| **シークレット** | 条件が隠されている実績です。ヒントを頼りに探してみましょう！ |

※プレイ時間は、この機能の独自で測るものになります。導入以前より参加していただいている方も0分スタートとなります。ご了承ください。

たくさんの実績解除を目指して、いねさばでの生活を楽しんでください！
