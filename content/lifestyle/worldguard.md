サーバー内での建築物を保護するために、土地の保護（Region）を設定する方法を解説します。

# 1. 準備するもの

土地の範囲を選択するために、**__木の棒__**を使用します。

# 2. 保護の手順

## ① 範囲を選択する
**木の棒**を手に持ち、保護したいエリアの対角線上の2点をクリックします。

1.  **左クリック** で始点（1点目）を指定
2.  **右クリック** で終点（2点目）を指定

::::details{title="範囲を垂直方向に拡張する（便利機能）"}
範囲を選択した後、以下のコマンドを入力すると、岩盤から空の高さまで垂直方向に範囲を拡張できます
:::command
//expand vert
:::
::::

## ② 保護領域を作成する
範囲選択ができたら、以下のコマンドを入力して保護を作成します。


:::command
/region create <自分の名前_保護名>
:::

:color[**必ず、保護名は「自分の名前_保護名(保護名は自由)」　にしてください！**]{class=text-red}（保護名の重複を避けるためです）

例：自分の家を `1necat_myhome` という名前で保護する場合
```
/region create 1necat_myhome
```

これで、自分以外のプレイヤーはその場所を編集できなくなります。

# 3. メンバーの追加（友達と共有する）

友達にも建築の権限を与えたい場合は、メンバーに追加します。

:::command
/region addmember <保護名> <プレイヤー名>
:::

例：`1necat_myhome` に `nekodesu` さんを追加する場合
```
/region addmember 1necat_myhome nekodesu
```

メンバーから削除する場合：
:::command
/region removemember <保護名> <プレイヤー名>
:::

# 4. 保護の削除

作成した保護を削除したい場合は、以下のコマンドを使います。

:::command
/region remove <保護名>
:::

# 5. 保護情報の確認

現在いる場所の保護情報を確認するには、以下のコマンドを使います。

:::command
/region info
:::
